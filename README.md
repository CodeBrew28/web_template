# micro-graphql
> Repo meant to be cloned for our GraphQL microservices

### Usage
+ Clone the repository
  ```bash
  git clone https://github.com/hyperfuse/micro-graphql.git
  ```

+ Install the dependencies
  ```bash
  yarn
  ```

+ Start the microservice
  ```bash
  yarn start
  ```

+ Make it your own!
